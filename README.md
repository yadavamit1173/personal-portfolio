# Personal Portfolio

A premium MERN-stack developer portfolio for **Amit Kumar Yadav**, designed to be inspired by modern developer portfolios while being more advanced, dynamic, and production-ready.

## Project Goal

Build a professional dark-themed portfolio platform with:
- a polished public portfolio UI
- MERN architecture (MongoDB, Express, React, Node.js)
- dynamic content management
- strong project and experience storytelling
- optional admin dashboard for future content updates

## Planned Stack

### Frontend
- React
- Vite
- Tailwind CSS
- React Router
- Framer Motion
- Axios

### Backend
- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT authentication
- Nodemailer or similar email integration

## Planned Features

### Public Portfolio
- Hero section
- About section
- Skills section
- Experience timeline
- Project showcase
- Education section
- Coding profiles and achievements
- Contact form

### Advanced Features
- Dynamic content from backend APIs
- Admin authentication
- Admin dashboard for managing portfolio content
- Contact form persistence
- Resume download support
- Project detail pages
- SEO-friendly structure
- Smooth animations and premium dark UI
